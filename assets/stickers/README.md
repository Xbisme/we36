# Story stickers (#005)

A small, fixed bundled sticker set for the Create Story overlay editor (US2, tasks T027).
Add lightweight PNG/SVG sticker assets here (kept intentionally tiny for v1.0 — research R5).
Declared in `pubspec.yaml` under `flutter/assets: assets/stickers/`.

Sticker keys are referenced from `StoryStickerOverlay.assetKey`
(`lib/features/stories/domain/models/story_sticker_overlay.dart`).
